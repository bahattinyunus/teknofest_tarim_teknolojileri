# Source Code (src/)

This directory contains the core source code for the Agri-Arch-TR project.

## Structure

```text
src/
├── iot/              # IoT sensor integration and LoRaWAN communication
├── ai/               # Machine Learning models for plant disease detection
├── backend/          # API and data processing services
└── automation/       # Autonomous vehicle control systems
```

## Getting Started

Each subdirectory contains its own README with specific setup instructions.
